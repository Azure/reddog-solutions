# virtual-customers
Based on [Virtual Customer](https://github.com/Azure/reddog-code/tree/master/RedDog.VirtualCustomers) from [reddog-code](https://github.com/Azure/reddog-code) repurposed for Java.
## How to run
### Prerequisites
- Java 11
- Maven 3
- Docker
- Docker Compose
- Dapr
## Run
- `mvn clean package`
- `docker-compose up`
