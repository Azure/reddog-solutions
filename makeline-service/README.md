# makeline-service
Based on [Makeline Service](https://github.com/Azure/reddog-code/tree/master/RedDog.MakeLineService) from [reddog-code](https://github.com/Azure/reddog-code) repurposed for Java.
## How to run
### Prerequisites
- Java 8
- Maven 3
- Docker
- Docker Compose
- Dapr
## Run
- `mvn clean package`
- `docker-compose up`
